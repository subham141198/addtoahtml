Description
The plugin adds a HTML snippet to a specific location on a webpage based on the selector name, target HTML, target page, and target location selected by the user.

Installation
Upload the plugin files to the /wp-content/plugins/html-injector directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.


Usage
Go to the 'Add to a Html' custom post type in the WordPress dashboard.
Add a new HTML snippet by filling out the selector name, target HTML, target page, and target location.
Save the post.
The HTML snippet will be added to the specified location on the target page.
